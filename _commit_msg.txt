feat(swipe): 拟真滑动全链路实现 + 变量节点 init action

核心能力:
- 新增 adbSwipe 两阶段架构：computeSwipeTrajectory 纯数学计算（Bezier + smoothstep 缓动 + 微抖动）+ 单次 ADB swipe 执行，避免分段短距被 Android 识别为 tap
- 15% 概率连续回弹（从终点连续第二次 swipe，不间断）

Server:
- libs/adb-swipe.ts: 轨迹计算 + 单次 adb shell input swipe 执行
- routes/device.routes.ts: POST /api/devices/swipe 接口
- engine.ts: execSwipe 方法 + swipe annotation 注解 + init action（变量不存在时才初始化，用于 session 跨轮持久化）

Web:
- flow/nodeTypes.ts: 新增 swipe 节点定义（startRegion/endRegion/padding）+ variable options 增加 init
- views/CanvasView.vue: 拟真滑动调试区 + SVG 轨迹叠加 + 动画 CSS
- components/ExecutionViewer.vue: swipe annotation SVG 路径渲染 + 起点/终点标记
- api/device.ts: deviceApi.swipe() 方法

Shared:
- types.ts: SwipePoint / AdbSwipeRequest / AdbSwipeResponse + VisualAnnotationType 增加 'swipe'
