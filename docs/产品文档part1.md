# Automan 产品文档 Part 1 — 已实现功能

> 本文档仅描述**已实现并可用**的功能，未实现的需求不在本文档范围内。

---

## 1. 产品定位

Automan 是一个面向模拟器自动化任务的控制面板。用户通过 Web 页面管理多台模拟器设备，使用开发板进行截图/圈图辅助操作。

**当前核心能力：**
- 多设备管理（CRUD，绑定雷电模拟器实例）
- WebSocket 实时连接（连接即登录，断开即退出）
- 开发板工具（设备截屏、框选坐标、缩放调整）

---

## 2. 核心概念

### 2.1 设备（Device）

设备是一个**任务合集入口**，绑定一台模拟器来运行。

| 属性 | 说明 |
|------|------|
| 别名 | 用户自定义名称，如"大号专用"、"小号专用" |
| 模拟器路径 | ldconsole.exe 文件的绝对路径 |
| 实例序号 | 雷电模拟器的 instance index |
| 状态 | **运行中** / **停止**（仅此两种） |

### 2.2 连接（Connection）

- **登录 = WebSocket 连接成功**
- 输入服务器地址（默认 `127.0.0.1`）和端口（默认 `3000`）
- WS 连接成功 → 自动跳转首页
- WS 断开 → 自动退出到登录页
- 已连接用户访问登录页 → 自动重定向到首页

## 4. 已实现页面

### 4.1 登录页（LoginView）

```
┌──────────────────────────────────────┐
│                                      │
│        ⚡ Automan                    │
│                                      │
│   服务器地址  [127.0.0.1      ]      │
│   端口        [3000            ]     │
│                                      │
│        [ 连接 ]                      │
│                                      │
└──────────────────────────────────────┘
```

- 全屏独立布局，不经过 AppLayout
- WS 连接成功后跳转首页
- 连接失败显示错误提示

### 4.2 首页 — 设备管理（HomeView）

```
┌─ aside (w-60) ─┐ ┌─ main (flex-1) ──────────┐
│ 设备列表   [+]  │ │                            │
│                │ │      设备详情区域            │
│ · 大号专用 ✏️🗑️│ │                            │
│ · 小号专用     │ │      （待扩展）             │
│                │ │                            │
└────────────────┘ └────────────────────────────┘
```

**左侧 aside（w-60）：**
- 设备列表，显示名称 + 运行状态图标
- hover 显示编辑（✏️）和删除（🗑️）按钮
- 右上角 link 样式「添加」按钮
- 选中高亮

**右侧 main：**
- 选中设备后显示设备信息（待扩展详情/任务配置）

**设备操作：**

| 操作 | 说明 |
|------|------|
| 添加 | 打开 DeviceDialog 弹窗 |
| 编辑 | 修改设备别名 |
| 删除 | 二次确认弹窗 |

### 4.3 开发板（CanvasView）

```
┌─ 操作区 (w-60) ─┐ ┌─ 截屏区 (flex-1) ──────────────────┐
│                  │ │                                      │
│ 当前设备 [▼]     │ │                                      │
│                  │ │          设备截图显示                  │
│ [ 截屏 ]         │ │          鼠标拖拽框选                  │
│                  │ │          顶点等比缩放                  │
│ 当前选中：  📋 ✕ │ │          边线单向缩放                  │
│ [x1,y1,x2,y2]   │ │          拖动移动选区                  │
│ 宽 W × 高 H     │ │                                      │
│                  │ │                                      │
│ [找图] [OCR]     │ │                                      │
│  ...             │ │                                      │
└──────────────────┘ └──────────────────────────────────────┘
```

**设备选择：**
- 下拉框，默认自动选中第一个设备

**截屏：**
- 点击「截屏」按钮，调用 ADB screencap
- 截图显示在右侧区域

**框选交互：**

| 操作 | 触发方式 | 说明 |
|------|----------|------|
| 新建框选 | 鼠标在截图区域拖拽 | 创建新的选区 |
| 移动选区 | 鼠标在选区内部拖拽 | 移动整个选区，实时更新坐标 |
| 等比缩放 | 拖拽四角圆形手柄 | 保持宽高比缩放 |
| 单向缩放 | 拖拽四边矩形手柄 | 仅改变宽度或高度 |
| 清除选区 | 点击选区外空白区域 | 或点击清除按钮（✕） |

**坐标回显：**
- 常显，不闪烁
- 默认 `[0,0,0,0]` 表示全屏，显示图片原始尺寸 + "(全屏)"标记
- 有选区时显示实际坐标 `[x1,y1,x2,y2]` + 选区尺寸
- 坐标基于原始分辨率换算（非显示像素）
- 复制按钮：点击后图标变 ✓（绿色），1 秒后恢复
- 清除按钮：有选区时显示，点击恢复 `[0,0,0,0]`

**视觉反馈：**
- 聚光灯蒙层：选区外半透明黑色遮罩
- 蓝色边框 + 白色 outline + 四角标记
- 实时宽高标注：黑底白字，上边线（宽）+ 右边线（高）
- 贴边吸附：拖动距边缘 < 5px 自动贴边

**Tab 区域：**
- 找图：上传目标图片（功能待实现）
- OCR：识别选区文字（功能待实现）

## 5. 设备添加弹窗（DeviceDialog）

```
┌─ 添加设备 ──────────────────────────────────────┐
│                                                  │
│  设备别名                                        │
│  [                    ]                          │
│                                                  │
│  ldconsole 路径                                  │
│  [                    ] [浏览]                   │
│                                                  │
│  ┌─ 文件浏览器 ────────────────────────────┐     │
│  │  📁 C:\                                 │     │
│  │  📁 D:\                                 │     │
│  │  📁 Program Files                       │     │
│  └─────────────────────────────────────────┘     │
│                                                  │
│  选择实例                                        │
│  ○ [0] 雷电模拟器-0 — 运行中                    │
│  ○ [1] 雷电模拟器-1 — 已停止                    │
│                                                  │
│              [取消]  [确认创建]                   │
└──────────────────────────────────────────────────┘
```

**交互流程：**
1. 输入设备别名
2. 点击「浏览」→ 打开文件浏览器，逐层导航选择 `ldconsole.exe`
3. 选中后自动加载模拟器实例列表（调用 `ldconsole list2`）
4. 单选目标实例（显示名称 + 运行状态）
5. 确认创建

## 6. API 路由清单

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/devices` | 查询设备列表 |
| POST | `/api/devices/create` | 创建设备 |
| POST | `/api/devices/delete` | 删除设备 |
| POST | `/api/devices/update` | 重命名设备 |
| POST | `/api/devices/instances` | 查询模拟器实例列表 |
| POST | `/api/devices/screenshot` | 设备截屏（ADB screencap） |
| GET | `/api/filesystem/browse?path=` | 浏览本地文件系统 |
| WS | `/ws` | WebSocket 连接 |

**统一响应格式：**
```typescript
// 成功
{ success: true, data: T }

// 失败
{ success: false, code: 'ERROR_CODE', message: '...' }
```

## 7. 技术栈

| 层级 | 技术 |
|------|------|
| 服务端 | Fastify + TypeScript |
| 数据库 | SQLite (better-sqlite3) + Drizzle ORM |
| 前端 | Vue 3 + TypeScript + UnoCSS + Vue Router + PrimeVue |
| 通信 | WebSocket (实时) + HTTP (REST API) |
| 图标 | primeicons CSS 类 |
| 包管理 | pnpm (monorepo) |

## 8. 项目结构

```
automan/
├── server/          # 服务端（Fastify）
│   └── src/
│       ├── routes/      # API 路由
│       ├── modules/     # 业务模块（device, ws, actor, task）
│       ├── db/          # 数据库（schema, migrate）
│       └── core/        # 核心（logger, event-bus）
├── web/             # 前端（Vue 3）
│   └── src/
│       ├── views/       # 页面（LoginView, HomeView, CanvasView）
│       ├── components/  # 组件（AppHeader, DeviceDialog）
│       ├── composables/ # 状态管理（useWebSocket, useDevices）
│       ├── api/         # HTTP 请求封装
│       ├── layouts/     # 布局（AppLayout）
│       └── router/      # 路由 + 守卫
└── shared/          # 前后端共享（types, utils）
    └── src/types.ts
```
