## 关键决策

1. **UnoCSS shortcuts 全部清除**，不再保留 `btn-primary` 等样式类，避免 PrimeVue 和 UnoCSS 样式冲突
2. **UnoCSS 保留用于布局**（flex、gap、padding 等工具类），与 PrimeVue 组件互补
3. **隐藏 file input 和 radio input 保持原生**，PrimeVue 无简化替代
4. **自定义 toggle switch 保持原生 CSS**（`wf-switch`），避免 PrimeVue InputSwitch 尺寸不匹配

## 遗留

- Logo 图片本身带有 "automan" 英文字样（品牌图形标识），UI 文案使用中文名"凹凸曼"
