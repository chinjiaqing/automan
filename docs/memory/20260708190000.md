# 工作总结 — 应用控制（启动/关闭/重启）

## 完成事项

在设备管理基础上，新增应用级控制能力：通过设备 ID + 应用包名，控制模拟器内应用的启动、关闭、重启。

## 新增 API

| 方法 | 路径 | Body | 功能 |
|------|------|------|------|
| POST | `/api/devices/:id/app/start` | `{ packageName }` | 启动应用 |
| POST | `/api/devices/:id/app/stop` | `{ packageName }` | 关闭应用 |
| POST | `/api/devices/:id/app/restart` | `{ packageName }` | 重启应用（先关等 1s 再开） |

## 变更文件

| 文件 | 变更 |
|------|------|
| `ldplayer.service.ts` | 新增 `runApp()` / `killApp()`（调用 ldconsole runapp/killapp） |
| `device.service.ts` | 新增 `startApp()` / `stopApp()` / `restartApp()` |
| `app.ts` | 新增 3 条应用控制路由 |
| `web/index.html` | 新增「📱 应用控制」面板（包名输入 + 三个按钮） |

## 底层命令

```
ldconsole runapp --index <n> --packagename <pkg>
ldconsole killapp --index <n> --packagename <pkg>
```
