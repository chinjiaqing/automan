## 关键决策

- **embeddable Python vs venv**：选 embeddable（11MB zip），免交互安装，比 venv 更适合分发
- **.cmd 启动 vs node.exe 直接调用**：pnpm 生成的 .cmd shim 内部处理 NODE_PATH，比手动调 node.exe 更可靠
- **get-pip.py 源**：华为云返回 HTML 错误页，改用 bootstrap.pypa.io
- **stdin error handler**：Python 进程崩溃时返回错误响应而非让服务 crash
