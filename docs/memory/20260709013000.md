## 三、框选交互增强

### 1. 拖动移动选区
- **触发**：mousedown 在已有选区内
- **模式**：`dragMode = 'move'`
- **行为**：拖动选区，实时更新坐标
- **约束**：不超出图片边界

### 2. 自动贴边吸附
```typescript
const SNAP_THRESHOLD = 5  // 像素

function snapToEdge(box) {
  if (box.left < SNAP_THRESHOLD) box.left = 0
  if (box.top < SNAP_THRESHOLD) box.top = 0
  if (imgW - (box.left + box.width) < SNAP_THRESHOLD) box.left = imgW - box.width
  if (imgH - (box.top + box.height) < SNAP_THRESHOLD) box.top = imgH - box.height
}
```

### 3. 实时宽高标注
```
        ┌──── 120px ────┐
        │               │
   80px │   选区内容     │
        │               │
        └───────────────┘
```
- **位置**：上边线居中（宽度）、右边线居中（高度）
- **样式**：黑底白字 `bg-black/75 text-white`
- **更新**：框选/拖动/缩放过程中实时显示

### 4. 顶点等比缩放（Corner Resize）
```typescript
// 四角圆形手柄
cornerHandles = [
  { id: 'tl', pos: { left: '0', top: '0' }, cursor: 'nwse-resize' },
  { id: 'tr', pos: { left: '100%', top: '0' }, cursor: 'nesw-resize' },
  { id: 'bl', pos: { left: '0', top: '100%' }, cursor: 'nesw-resize' },
  { id: 'br', pos: { left: '100%', top: '100%' }, cursor: 'nwse-resize' },
]
```
- **样式**：蓝色圆形 + 白色边框 (`w-3 h-3 rounded-full`)
- **行为**：保持宽高比缩放，对角锚点不动
- **约束**：最小尺寸 10px，不超出图片边界

### 5. 边线单向缩放（Edge Resize）
```typescript
// 四边中点矩形手柄
edgeHandles = [
  { id: 'top', cursor: 'ns-resize' },      // 仅改高度
  { id: 'bottom', cursor: 'ns-resize' },   // 仅改高度
  { id: 'left', cursor: 'ew-resize' },     // 仅改宽度
  { id: 'right', cursor: 'ew-resize' },    // 仅改宽度
]
```
- **样式**：白色矩形 + 蓝色边框 (`24px × 6px`)
- **行为**：仅改变单方向尺寸
- **约束**：最小尺寸 10px，不超出图片边界

## 四、状态管理

### DragMode 枚举
```typescript
type DragMode = 'none' | 'new' | 'move' | 'resize'
```

| 模式 | 触发条件 | 行为 |
|------|----------|------|
| `new` | mousedown 在容器空白处 | 新建框选 |
| `move` | mousedown 在选区内部 | 拖动移动选区 |
| `resize` | mousedown 在缩放手柄上 | 缩放选区 |
| `none` | mouseup | 空闲状态 |

### 缩放类型
```typescript
type HandleType = 'corner' | 'edge'
type HandleId = 'tl' | 'tr' | 'bl' | 'br' | 'top' | 'bottom' | 'left' | 'right'
```

const scaleX = screenshot.width / imgDisplaySize.width

selection = {

  y1: Math.round(box.top * scaleY),

  y2: Math.round((box.top + box.height) * scaleY),

### 2. 等比缩放算法
```typescript
function calcProportionalResize(dx, dy) {
  const ratio = init.width / init.height
  // 根据角点方向确定主方向 delta
  let delta = Math.max(dx, dy)  // br 角
  // ... 其他角点方向不同

  let newW = init.width + delta
  let newH = newW / ratio  // 保持宽高比

  return { left, top, width: newW, height: newH }
}
```
