## 职责划分原则

- **LDPlayerService**：只管 `ldconsole.exe` 命令
- **AdbService**：只管 `adb.exe` 命令（connect / screencap / 未来更多 ADB 操作）
- **DeviceManager**：编排层，组合 ld + adb，不直接执行子进程
- **routes/**：HTTP 路由层，保持薄层，只做参数提取 → 调用 service → 返回响应
- **app.ts**：只做编排（注册插件 → 初始化服务 → 挂载路由），不包含业务逻辑
